# Reglas de ORO — Mis Quiero en Acción

Estas reglas aplican a **todas** las interacciones y cambios del proyecto.

## 1) Paso a paso (sin suponer conocimiento)
- Se avanza **de a un paso por vez**.
- Cada paso incluye:
  - **Qué hacer**
  - **Resultado esperado**
  - **Cómo verificar**
- No se asume conocimiento técnico previo.
- Si falta un dato, se pregunta **una sola cosa** y se detiene el avance hasta tenerlo.

## 2) Archivos completos (sin parciales)
- Cuando se entregue un archivo, se entrega **completo** (contenido íntegro).
- No se envían fragmentos para copiar y pegar.
- Siempre se indica:
  - **Ruta exacta del archivo**
  - **Contenido completo**
  - **(Opcional) comando de reemplazo/verificación**, si corresponde

---

## Cómo usar estas reglas en un nuevo chat
Pegá esta frase al inicio del chat:

> “Aplican las Reglas de ORO del proyecto (ver `reglas.md`).”
